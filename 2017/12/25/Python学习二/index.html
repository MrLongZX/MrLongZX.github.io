Python学习二

输入输出

数据类型

整形：精确，没有大小限制， / 除法计算结果是浮点数，即使是两个整数恰好整除，结果也是浮点数

还有一种除法是 // ，称为地板除，两个整数的除法仍然是整数：

取余：

浮点数：有四舍五入，没有大小限制

字符串：转义字符 \ ，\n表示换行，\t表示制表符，\表示字符\。

布尔值：只有True、False，布尔值可以用and（与）、or（或）和not（非）运算

空值：None

变量

可以是任意数据类型，必须是大小写英文、数字和_的组合，且不能用数字开头。

可以把任意数据类型赋值给变量，同一个变量可以反复赋值，而且可以是不同类型的变量。

常量

常量就是不能变的变量，通常用全部大写的变量名表示常量
