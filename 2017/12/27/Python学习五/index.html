Python学习五

dict

字典，全称dictionary，在其他语言中也称为map，使用键-值（key-value）存储，具有极快的查找速度。

dict内部存放的顺序和key放入的顺序是没有关系的。

dict的key必须是不可变对象。

set

也是一组key的集合，但不存储value。由于key不能重复，所以，在set中，没有重复的key。

重复元素在set中自动被过滤。

无序和无重复元素的集合。

可以做数学意义上的交集、并集等操作。
